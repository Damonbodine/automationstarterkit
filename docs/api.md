# API Reference

> Consolidated from `docs/00-api-specs.md`. Endpoints reflect Next.js API routes under `src/app/api/**`.

- Auth: `/api/auth/[...nextauth]` (NextAuth managed)
- Emails: `/api/emails`, `/api/emails/[emailId]`, classification, attachments, OCR, bulk endpoints
- Agents: `/api/agents/queue`, `/api/agents/logs/recent`
- Projects/Tasks/Documents: `/api/projects/**`, `/api/tasks/**`, `/api/documents/**`
- Webhooks: `/api/webhooks/gmail`
- Queues: `/api/queues/stats`
- Health: `/api/health/integrations`
